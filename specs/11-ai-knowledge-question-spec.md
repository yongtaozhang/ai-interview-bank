# AI Knowledge Question Spec

## 1. Goal

为前端面试宝典项目新增一批 AI 知识相关面试题。

本 Spec 的目标是补充适合前端开发者、资深前端、AI Coding 项目展示场景的 AI 基础知识题，而不是转向纯算法工程、模型训练或深度学习研究题。

新增题目应帮助用户理解：

- LLM 基础概念
- Prompt Engineering
- Token 和上下文窗口
- Embedding 和向量检索
- RAG
- Agent
- Tool Calling / Function Calling
- MCP
- AI 评估
- AI 安全
- AI Chat 前端工程实践
