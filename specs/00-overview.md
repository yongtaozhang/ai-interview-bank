# AI 前端面试宝典 MVP 总览

## 项目一句话

AI 前端面试宝典是一个面向资深前端面试准备的速查型 MVP，用于快速检索前端面试题、查看口述答案、关键点、追问方向，并沉淀一页 AI Coding / Codex Prompt 模板。

## 项目目标

1. 为前端开发工程师提供一个高效、结构化、可快速浏览的面试题库。
2. 支持按关键词、分类、难度、高频程度快速定位题目。
3. 在题目详情页提供适合面试口述的答案、关键点、追问与延展方向。
4. 作为 AI Coding 实践项目展示，体现 Spec Coding、清晰任务拆分和可验收交付。
5. 第一版以纯前端静态 MVP 为目标，优先保证信息架构、检索体验和内容展示质量。

## MVP 范围

### 做什么

- 首页展示项目定位、题库入口、主要分类和高频题入口。
- 题库页展示结构化面试题列表。
- 支持关键词搜索。
- 支持分类筛选。
- 支持难度筛选。
- 支持高频程度筛选。
- 题目详情页展示完整题目信息。
- 支持一键复制口述答案。
- AI Coding 页面展示 Codex Prompt 模板。
- 使用本地静态数据作为题库来源。

### 暂不做什么

- 不做登录、注册、用户系统。
- 不做后端 API。
- 不做数据库。
- 不做收藏。
- 不做刷题流程。
- 不做掌握状态。
- 不做学习统计看板。
- 不做题目编辑后台。
- 不做在线同步。
- 不做付费、分享、评论等社区能力。

## 目标用户

主要用户是一名前端开发工程师，正在准备资深前端面试，同时希望通过该项目展示 AI Coding 的工程实践能力。

## 核心用户价值

- 快速找到某个方向的典型面试题。
- 快速组织一段可口述的高质量答案。
- 通过关键点和追问查漏补缺。
- 用结构化项目展示前端产品、工程、交互和 AI Coding 能力。

## 技术栈约束

- React
- Vite
- TypeScript
- React Router
- Tailwind CSS
- Fuse.js
- Lucide React

## 核心路由

- `/`：首页
- `/questions`：题库页
- `/questions/:id`：题目详情页
- `/ai-coding`：AI Coding 页面

## 成功标准

- 用户能在 30 秒内从首页进入题库并找到感兴趣的题目。
- 用户能通过关键词和筛选组合缩小题目范围。
- 用户能在详情页获得可直接练习口述的答案。
- 用户能一键复制口述答案。
- 项目结构清晰，后续可自然扩展登录、收藏、后端或数据库。
