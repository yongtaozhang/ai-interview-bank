## 路由验收

- [ ] `/` 能访问首页。
- [ ] `/questions` 能访问题库页。
- [ ] `/questions/:id` 能访问对应题目详情页。

## 数据验收

- [ ] 每道题有唯一 `id`。
- [ ] 每道题有标题、摘要、分类、难度、高频程度。
- [ ] 每道题至少有一个标签。
- [ ] 每道题有口述答案。
- [ ] 每道题有关键点列表。
- [ ] 每道题有追问字段，即使为空数组。
- [ ] 关联题目 ID 都能在题库中找到。
- [ ] 搜索字段覆盖标题、摘要、标签、分类、答案和关键点。

## 手动测试建议

1. 打开首页，确认导航和入口正常。
2. 进入题库页，确认默认展示全部题目。
3. 输入关键词，确认结果变化。
4. 切换分类、难度、高频程度，确认结果符合所有条件。
5. 清空筛选，确认列表恢复。
6. 点击题目卡片，确认进入详情页。
7. 点击复制口述答案，确认有成功反馈。
8. 访问不存在的题目 ID，确认错误状态友好。
9. 进入 AI Coding 页面，确认模板展示和复制正常。
10. 使用移动端视口检查首页、题库页和详情页。
